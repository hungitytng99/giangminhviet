## LikesTodays
A web application of Nobisoft company </br>
To start project: </br>
1. `npm i`
2. `npm run dev`
## WARNNING:
1. Before run web, must disable CORS on browser by link: https://alfilatov.com/posts/run-chrome-without-cors/
- Description: Make new chrome with none CORS.

## Available route
1. `/` or `/home`
